Quick Start

1Prerequisites

Make sure you have these installed:

  • Node.js (v16 or higher) - Download here
  • Git - Download here
  • A Firebase account (free) - Create here
  • A Vercel account (free) - Sign up here

2Install Dependencies

Open terminal in the project folder and run:

npm install

3Set Up Firebase

Create a Firebase project and configure it:

  1. Go to Firebase Console
  2. Create a new project
  3. Enable these services:
    • ‚úì Firestore Database
    • ‚úì Authentication (Email/Password)
    • ‚úì Storage
  4. Get your Firebase configuration
  5. Create an admin user in Authentication
üìñ Detailed Instructions: See FIREBASE_SETUP.md for step-by-step guide with screenshots.

4Configure Firebase in Your App

Open app.js and replace the Firebase configuration (around line 23):

const firebaseConfig = { apiKey: "YOUR_API_KEY", authDomain: "YOUR_PROJECT_ID.firebaseapp.com", projectId: "YOUR_PROJECT_ID", storageBucket: "YOUR_PROJECT_ID.appspot.com", messagingSenderId: "YOUR_MESSAGING_SENDER_ID", appId: "YOUR_APP_ID" };
‚ö†Ô∏è Important: Replace ALL placeholder values with your actual Firebase credentials!

5Run Locally

Start the development server:

npm run dev

Open your browser to: http://localhost:5173

6Test the Application

  • Submit a test membership application
  • Check if data appears in Firestore
  • Verify photos are uploaded to Storage
  • Login to admin panel with your credentials
  • View, edit, and delete test applications

7Deploy to Vercel

Deploy your application to the internet:

npm i -g vercel vercel login vercel --prod

Your app will be live at: https://your-app-name.vercel.app
